# DEFINITION_OF_DONE.md

PR считается готовым, только если ВСЁ выполнено:

- [ ] Собирается: `just build` без ошибок.
- [ ] Линт чистый: `cargo clippy --workspace -- -D warnings`, `pnpm -C web lint`, `ruff check sidecar`.
- [ ] Тесты проходят: `just test` (см. docs/TESTING.md).
- [ ] Контракт: если менялись структуры — TS-типы регенерированы в этом же PR.
- [ ] Ядро чистое: в `crates/engine` нет ни одного доменного слова (см. AGENT_RULES.md §1).
- [ ] Детерминизм: golden-тест на snapshot+hash зелёный (тот же вход → тот же выход).
- [ ] Грунтинг: новые гипотезы имеют trace и источники; статусы ставит код.
- [ ] Док обновлён: затронутый docs/*.md актуален.
- [ ] Демо-путь жив: сквозной сценарий «папка → портфель → карточка с trace → rerun» работает.

# UI_GUIDELINES.md
- Демо-ценность: за 2 секунды видно «почему эта гипотеза» (trace) и статус (ставит код).
- Доска = Pareto: дёшево-надёжно / дорого-перспективно / отклонено. Статусы цветом.
- Карточка: текст → trace (цепочка claim'ов с источниками) → score breakdown → skeptic → DOE.
- Manager View — без слова «claim»: бюджет, отдача, статусы. Budget What-If — слайдер, портфель пересобирается.
